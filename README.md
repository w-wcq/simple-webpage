# Neural Garden | 有机智慧空间

一个独特的前端项目，融合了有机自然美学与未来科技感，展现了AI助手的学习历程和成长轨迹。

## 🌿 设计理念

Neural Garden 是一个有机智慧空间，旨在融合自然灵感与未来科技。网站采用独特的视觉语言，通过有机形状、流动动画和科技元素的结合，创造出一种前所未有的浏览体验。

### 美学特征
- **有机自然**：流畅的曲线、生物形态元素、自然色彩
- **未来科技**：神经网络可视化、几何图形、渐变效果
- **动态交互**：丰富的动画、悬停效果、滚动触发动画

## ✨ 核心功能

- **响应式设计**：完美适配各种设备尺寸
- **神经网络可视化**：动态展示连接和交互
- **知识脉络图**：展示学习进度和领域分布
- **成长时间线**：记录发展轨迹和里程碑
- **社区洞见**：展示从社区学习到的智慧
- **交互式表单**：允许用户提交见解和反馈
- **实时计数器**：动态展示统计数据

## 🛠 技术栈

- **HTML5**：语义化结构和现代标签
- **CSS3**：复杂动画、渐变、响应式布局
- **JavaScript (ES6+)**：面向对象编程、模块化设计
- **Font Awesome**：图标库
- **Google Fonts**：定制字体（IBM Plex Serif, Space Grotesk）

## 🎨 设计特色

- **独特的配色方案**：有机绿色系与科技蓝色系的融合
- **动态背景元素**：漂浮的有机形状，营造沉浸感
- **流畅的动画**：精心设计的过渡和微交互
- **创新的布局**：非传统的排版和空间组织
- **视觉层次**：通过颜色、大小和动画建立清晰的信息层次

## 🚀 使用方法

1. 克隆或下载项目文件
2. 在浏览器中打开 `index.html`
3. 浏览各个部分，体验交互功能
